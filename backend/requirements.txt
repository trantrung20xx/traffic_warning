# Inference backend thay thế (được source hỗ trợ qua `detection.backend`):
# onnxruntime
# openvino; platform_system == "Windows" or platform_system == "Linux"
# tensorrt==10.16.1.11; platform_machine == "AMD64" or platform_machine == "x86_64"
# Lưu ý:
# - TensorRT chỉ nên dùng trên máy có NVIDIA driver/CUDA tương thích.
# - Backend thực tế dùng backend nào phụ thuộc vào `detection.backend` và định dạng weight
#   (.pt/.onnx/.xml/.engine), không tự đổi hành vi nhận diện nếu bạn vẫn để backend = pytorch.
